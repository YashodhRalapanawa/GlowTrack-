# GlowTrack

GlowTrack is a comprehensive wellness mobile application designed to help users build and maintain healthy habits while tracking their emotional well-being and hydration levels. Built with Kotlin for Android, it provides an intuitive interface for personal growth and self-care.

## Features

### 🔐 User Authentication
- Secure login and registration system
- Persistent user sessions with SharedPreferences

### 📱 Habit Tracking
- Create custom habits with personalized targets (e.g., exercise, reading, meditation)
- Set reminder notifications for habit completion
- Track progress with visual indicators and statistics
- Support for various units: times, minutes, hours, steps, pages, etc.
- Quick habit creation via bottom sheet interface

### 😊 Mood Journaling
- Daily mood tracking with emoji-based selection
- 10 different mood types: Very Happy, Happy, Excited, Calm, Neutral, Tired, Anxious, Sad, Angry, Very Sad
- Add personal notes to mood entries
- View mood history and trends
- Enhanced mood tracking with detailed analytics

### 💧 Hydration Monitoring
- Set personalized daily water intake goals (500ml - 5000ml)
- Log hydration intake with predefined amounts (glasses, bottles)
- Customizable reminder intervals (30min - 4 hours)
- Hydration history and progress visualization
- Smart scheduling between customizable start/end times

### 🔔 Smart Notifications & Reminders
- Exact alarm scheduling for habit and hydration reminders
- Notification permissions for timely alerts
- Vibration and snooze options for reminders
- Boot-completed receiver to restore alarms after device restart

### 📊 Analytics & Insights
- Mood trend charts and statistics
- Habit completion streaks and consistency tracking
- Hydration progress visualization
- Best day tracking and milestone indicators

### ⚙️ Settings & Customization
- App preferences and configuration
- Backup and data management
- Theme customization options

### 🎨 User Interface
- Modern Material Design 3 interface
- Bottom navigation for easy access to main features
- Smooth animations and transitions
- Responsive design for various screen sizes

## Technical Features

- **Platform**: Android (API 21+)
- **Language**: Kotlin
- **Architecture**: MVVM with Fragments
- **Data Storage**: SharedPreferences for local data persistence
- **Background Tasks**: WorkManager for hydration reminders
- **Sensors**: Activity recognition for enhanced tracking
- **Widgets**: Home screen widget for habit progress
- **Permissions**: Notifications, exact alarms, activity recognition, vibration

## Getting Started

1. **Installation**: Download and install the APK on your Android device
2. **Onboarding**: Complete the initial setup process
3. **Authentication**: Create an account or log in
4. **Setup**: Configure your habit goals, hydration targets, and preferences
5. **Track**: Start logging your habits, mood, and water intake
6. **Monitor**: View your progress through charts and statistics

## Privacy & Security

- All data is stored locally on your device
- No internet connection required for core functionality
- User data remains private and secure
- Optional backup features for data preservation
